# OptiFlow AI – Next.js website

Dit is een kant-en-klaar Next.js project voor je homepage.

## Lokaal starten

1. Installeer Node.js 20 of nieuwer
2. Open de map in VS Code
3. Run:

```bash
npm install
npm run dev
```

Open daarna:

```bash
http://localhost:3000
```

## Online zetten met Vercel

1. Maak een GitHub repository aan
2. Upload deze projectbestanden
3. Ga naar Vercel
4. Import je GitHub repository
5. Klik op Deploy
6. Koppel daarna je domein

## Bestanden
- `app/page.tsx` → homepage
- `app/layout.tsx` → basis layout + metadata
- `app/globals.css` → globale styling
